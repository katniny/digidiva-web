Privacy Policy Updated: Feb 24, 2025


Katniny Studios is committed to privacy and transparency in how DigiDiva, our Discord app, handles data.

This Privacy Policy explains what data DigiDiva collects, how it's used, and what control server owners and users have over it.


What Data DigiDiva Collects

DigiDiva only collects and stores the following:

  • Conversation Data (AI Memory): When using DigiDiva's AI features, she stores conversation data, which includes:
  • Message content
  • Sender's display name
  • Role (either "user" or "assistant", where "assistant" is always DigiDiva)
  • Server Settings: If server administrators configure settings using commansd (e.g. /settings enableconfessions), those preferences are stored for functionality.

    • DigiDiva does not store personally identifiable information (PII) beyond what is listed above.


Moderation & Logging

  • Moderation Checks: DigiDiva may scan messages to enforce moderation rules and detect when she is mentioned for AI interactions.
  • Logging: Server administrators can enable logging features (e.g. moderation actions). This may include message content and user IDs, but logs are only sent to the designated logging channel and are not stored by DigiDiva.

How Data is Used

  • AI memory helps DigiDIva provide contextual responses over time, and is never used to train our AI models.
  • Server settings ensure DigiDiva functions according to admin preferences.
  • Moderation tools help maintain a safe server environment.

Who Has Access to This Data?

Only DigiDiva (operated by Katniny Studios) has access to stored AI conversation data and server settings. We do not share, sell, or distribute collected data to anyone.


Your Control Over Data

  • Admins can modify settings at any time.
  • AI memory is persistent, meaning stored conversation data is retained.

By using DigiDiva, you agree to this Privacy Policy. If you have any questions, contact @katniny on Discord.
